React Native入门相关

发表于 2018-06-20

下一阶段的业余计划:

不知怎么的,想写写最近发生的事,就当记录下流水账吧,终于我开启了在HK的996面试,说实话,我有点疲惫,有各种因素的加班,可能是业务没做完,也有可能是部门的因素,已经养成了加班的风气,毕业两年的时间,突然有点迷惘,原来大公司也是这样,说实话我有点不知道之后的路究竟该怎么走了,觉得自己现阶段好迷惘啊,前段时间给自己定几个目标:

  1. 参与公司一个项目的重构，认领到自己相应的模块,尽力去完成;
  2. 学习rn相关，把自己很久以前的ionic项目重构成rn项目;
  3. 学习koa2,尝试着从最开始的注册,登录开始自己搭建一个博客出来；
    说实话,重构那个项目我做的有限，当时因为喜欢一个姑娘,当时和旁边几个人相反设法的在要她的联系方式,在加上前段时间自己确实不想写代码,心态方面出现了很大的问题,在实际维护这个项目的过程中也没有深入到业务的痛点问题中去,只是单纯把之前项目的redux给去掉,通过用state和props的方式去实现而已,感觉实际上对自己的提升帮助有限；还有在维护赣州这个老项目的过程中,心态差点崩掉,进去的时候踌躇满志结果整天就是在维护这个糟心的jquery项目,说实话,学技术从功利角度我还是想学新的东西的,对扩充人的技术视野有很大的好处,虽然说基础很重要,但是基础这个东西不是一朝一夕可以提升上去的,所以我觉得这个东西还是慢慢的提升自己吧，充分利用自己身上的优势,去做到超越别人.
    这段时间参加了node party，觉得现阶段还是把自己的基础打好是最核心的,所以现阶段node的计划还是暂且缓一下,可以尝试着深入react方向,业余有时间把vue,webpack学一下,然后再有机会再去深入react native,rx.js等。
    还了解到了有关服务端ssr渲染相关的,还了解了react native,自己尝试做了简单的demo，通过与一些同行的交流,充分认识到了自身还有很多不足,自身还需要去提高的地方,必须保持着一颗谦卑的心,尽力去做好自己当下的工作,最大化的提高自己的代码水平.

ps:现阶段对rn的了解的话暂时卡在调试和分离两个步骤,自己最近用安卓做模拟的环境的时候感觉比ios环境上更坑。
