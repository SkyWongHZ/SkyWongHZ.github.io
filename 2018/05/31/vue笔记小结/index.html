1.vue项目部署在webstorm中，第一个遇到的问题是，webstorm卡住了，没错，大卡特卡。原因是webstorm在后台检索索引，而我们的vue项目中node_modules下包会很多，所以会很卡，解决方式是让webstorm停止对node_modules检索，右键node_modules，选择Mark Directory As为Excluded，卡顿消失。

以及因为使用IDE打开Vue项目会给node_modules创建索引，而node_modules文件庞大，导致创建索引过慢，于是笔者提供的思路就是将node_modules忽略，不为其创建索引。
